#### 接口说明 1、获取课件详情

- **请求URL**
> [courseManager/findCourseInfo](#)

- **请求方式**
>**GET**

- **请求参数**

| 请求参数      |  参数类型 |  参数说明     |
| :--------     | :-------- | :------       |
| id     | String    |  课件id |


- **返回**

| 返回参数  |     参数类型  |   参数说明                            |
| :-------- | :--------     | :------                               |
| info      | Array         | 返回数据集合                          |
| code      | Number        | 状态码， 系统状态码， 与http状态码区别|
| status    | Boolean       | 状态true/false, 操作是否成功          |


- **返回示例**


```javascript
{
  "info": [
    {
        
    }
  ],
  "code": 200,
  "status": true
}
```
